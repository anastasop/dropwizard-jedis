Welcome

This is a demo application for dropwizard-jedis, a bundle which integrates dropwizard and jedis a popular redis client.

The concept is simple. A set of teams and their members are stored in redis. These are all manipulated using the following restful urls. Of course redis must be up and running before using them. You can check this using the dropwizard admin console 

GET   /teams/
GET   /teams/{name}
POST  /teams/{name}
These urls work with both html and json. For example to add a team and its initial members use 
curl -H"Content-Type: text/plain" -d"Balotelli,Kaka" http://localhost:8080/teams/Milan
You can access the database using the following link. Teams
